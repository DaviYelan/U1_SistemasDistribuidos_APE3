# U1_SistemasDistribuidos_APE3

Para arrancar el programa:

1) Instalar dependencias (una sola vez):

cd Proyecto_APE3_Distribuidos/Backend

pip install -r requirements.txt

2) Levantar el backend:

python app.py

3) Levantar el frontend:

python server.py

Abrir el frontend desde el localhost 3000
